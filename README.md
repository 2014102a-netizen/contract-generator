# contract-generator
Генератор договоров для предмета Код. Текст. Дизайн
Автоматическая генерация договоров подряда и оказания услуг, создано на базе шаблонов и для нужд строительной компании "Альянс", а также любых существующих организаций, использующих для коммерческой коммуникации договорные отношения, в рамках Российского правового поля. 

Авторы и выполненные задачи
Андрей - модуль работы с файлам (file_handler.py) , модуль валидации данных (data_validator.py), консольный интерфейс (main_console.py),  интеграция всх модулей, настройка Github, предоставление шаблонов договоров.

Дмитрий  - класс Contract (contract_generator.py), , архитектура системы генерации, ООП структура проекта, проверка и тестирование.

Николай - GUI графический интерфейс ( main_gui.py), дизайн пользовательского интерфейса, UX-проектирование, разрабтока принципа работы программы, оптимизация приянтых технических решений, генератор тайм менеджемента команды. 

Описание

Программа для автоматического создания договоров на основе шаблонов.

Возможности:
Консольный интерфейс (main_console.py)
Графический интерфейс (main_gui.py)
Валидация данных (ИНН, даты, суммы)
Сохранение истории договоров

Структура проекта

contract-generator/

file_handler.py          # Работа с файлами

data_validator.py        # Валидация данных

contract_generator.py    # Генерация договоров (класс Contract)

main_console.py          # Консольный интерфейс

main_gui.py              # Графический интерфейс

templates/               # Шаблоны договоров

договор_подряда.txt

договор_услуг.txt

Установка и запуск

Требования

Python 3.8+

Запуск консольной версии

терминал:
python main_console.py
Запуск GUI версии
термиал:
python main_gui.py

Примеры использования
Создание договора подряда

1. Запустите программу
2. Выберите тип договора
3. Введите данные
4. Проверьте и подтвердите
5. Договор сохранён в папку `output/`
